# xlimport-app

## Run this code to launch the project

```bash
git clone https://github.com/MikhailKlidzhan/xlimport-app.git
cd xlimport-app

# create .env with this data:
# DB_NAME=xlimportdb
# DB_USER=admin
# DB_PASSWORD=admin
# DB_HOST=db
# DB_PORT=5432

docker-compose up --build

# run on localhost:8000/
```
